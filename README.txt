SITE - ACADEMIA BARROWMONT

Como usar:
1. Extraia a pasta site_barrowmont.zip.
2. Abra o arquivo index.html no navegador.
3. Para editar textos, altere o index.html.
4. Para editar cores e visual, altere o style.css.
5. Para editar interações, altere o script.js.

Estrutura:
- index.html: conteúdo principal do site.
- style.css: visual do site.
- script.js: menu, filtros e animações.
- assets/: brasão e favicon em SVG.

Observações:
Este site foi criado como página inicial estática para o RPG Academia Barrowmont.
Ele pode ser hospedado em GitHub Pages, Netlify, Vercel ou qualquer hospedagem simples.
